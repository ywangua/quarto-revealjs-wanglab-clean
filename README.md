# Quarto clean theme

This is a fork of [grantmcdermott/quarto-revealjs-clean](https://github.com/grantmcdermott/quarto-revealjs-clean).

* Text colors and some styles are changed to fit my personal taste.
* A few styles are added.

## Use

```bash
quarto install extension ywangua/quarto-revealjs-clean
```

```bash
quarto use template ywangua/quarto-revealjs-clean
```
